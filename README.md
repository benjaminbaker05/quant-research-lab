# Quant Research Lab

Research project looking at short-horizon prediction, market microstructure, market making, and transaction-cost-aware backtesting.

## Goals

- Build a clean Python research pipeline for market data.
- Implement statistical and machine learning models for short-horizon prediction.
- Build a C++ limit-order-book simulator
- Evaluate strategies under realistic transaction costs and risk constraints.
- Producea  technical research report explaining assumptions, results, and failure modes.

## Core Skills Demonstrated

- Python research engineering
- C++ performance programming
- Machine learning
- Probability and statistics
- Market microstructure
- Backtesting and model evaluation
- Reproducible research
